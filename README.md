# android Calculator

a minimal yet powerful calculator application for Android, built with **Kotlin**.

## features

- supports basic arithmetic operations: addition, subtraction, multiplication, and division.
- parentheses support for complex expressions.
- decimal point handling with proper validation.
- real-time evaluation using `exp4j`.
- simple and intuitive UI.
